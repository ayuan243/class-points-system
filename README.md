# 🏆 班级积分系统

> 适合老师日常使用的班级积分管理工具，支持手机实时查看积分。

---

## ⬇️ 下载使用（推荐）

**Windows 用户直接下载 exe，双击运行，无需安装任何软件：**

👉 **[下载 ClassPoints-v1.0.exe](https://github.com/ayuan243/class-points-system/releases/download/v1.0/ClassPoints-v1.0.exe)**

> Mac / 其他设备：下载 `班级积分系统.html`，用浏览器打开即可使用。

---

## 🚀 快速开始

1. 下载并运行 `ClassPoints-v1.0.exe`
2. 进入「**管理**」页 → 点「**批量导入**」→ 粘贴学生名单（每行一个名字）→ 导入
3. 回到首页，点击学生头像即可加分

## 📱 手机实时查看配置

1. 注册 [Firebase](https://console.firebase.google.com)（免费，需要 Google 账号）
2. 新建项目 → 左侧点 **Realtime Database** → 创建数据库 → 选「测试模式」
3. 复制数据库 URL（如 `https://xxx.firebasedatabase.app`）
4. 在程序「**设置 → 手机实时查看**」里粘贴该地址，保存
5. 手机打开 `班级积分系统_手机查看.html`，输入同一地址即可实时查看

---

## 💾 数据说明

- 所有数据保存在**本机本地**，不会上传到任何服务器
- 开启云同步后，数据同步至你自己的 Firebase 数据库（仅你可控）
- 重装系统或换电脑时，可通过「管理 → 导出数据」备份

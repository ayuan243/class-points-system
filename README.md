# 🏆 班级积分系统

一个本地运行的班级积分管理工具，无需安装，直接用浏览器打开即可使用。

## 文件说明

| 文件 | 说明 |
|------|------|
| `班级积分系统.html` | 主程序（老师端），双击用浏览器打开 |
| `班级积分系统_手机查看.html` | 手机查看端，配合 Firebase 实时查看积分 |
| `班级积分系统_class 5.html` | 五班专用抽签工具 |

## 功能

- 🧑‍🎓 学生积分管理，支持批量导入
- ➕ 点击头像即可加分，支持自定义分值
- 🎲 随机抽人（不重复模式）
- 📊 个人/班级统计，本周 vs 上周对比
- 📤 一键导出积分报表（可打印）
- 🔊 加分音效
- ☁️ Firebase 云同步，手机实时查看
- 📱 手机端查看积分排行榜 / 记录

## 使用方法

1. 下载 `班级积分系统.html`，用浏览器打开
2. 在「管理」页导入学生名单
3. 在「设置 → 手机实时查看」配置 Firebase 地址（免费）
4. 将 `班级积分系统_手机查看.html` 发给需要查看的人

## 云同步配置

1. 注册 [Firebase](https://console.firebase.google.com)（免费）
2. 创建项目 → Realtime Database → 测试模式
3. 复制数据库 URL 填入设置页面

---

数据保存在本地浏览器（localStorage），不会上传到任何服务器。

☁️ 手机实时查看

使用免费的 Firebase 数据库，加分后手机可实时看到数据。
第一次使用？按下方步骤获取链接（约 3 分钟）。
Firebase 数据库地址
粘贴你的 Firebase Realtime Database URL（以 .firebaseio.com 结尾）
📋 如何获取 Firebase 链接（免费）
  1. 打开 console.firebase.google.com，登录 Google 账号
  2. 点击「新建项目」，随便起个名字
  3. 左侧菜单点 Realtime Database → 「创建数据库」
  4. 选「以测试模式启动」→ 完成
  5. 复制页面顶部的数据库 URL（如 https://xxx.firebaseio.com）
  6. 粘贴到上方输入框，点保存
📤 其他电脑 / 手机查看
点击生成一个网页，任何设备用浏览器打开就能看积分，不需要传文件。
打开后在浏览器收藏该页面，下次直接从收藏夹访问即可。
